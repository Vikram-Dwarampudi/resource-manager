Key Workflows:
--------------------
1. View Resources: User → Frontend → GET /servers → Backend → JSON → Display
2. Create Pod: User → Form → POST /create → Validate → Update JSON → Deploy K8s
3. Update Resources: User → Edit → POST /update → Check → Update → Sync K8s
4. Delete Pod: User → Confirm → POST /delete → Remove JSON → Delete K8s

Technology Stack:
--------------------
• Frontend: Angular 20.1.0 with Material Design
• Backend: Flask with CORS, Swagger documentation
• Data: JSON file-based storage (mock_db.json)
• Container: Kubernetes integration
• API: RESTful endpoints with JSON payloads
